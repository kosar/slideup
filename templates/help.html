Introduction to SlideUp

SlideUp is a powerful set of tools designed to streamline the creation and enhancement of PowerPoint presentations. The project consists of two main applications:

  1. Markdown to PPTX Converter: Create professional presentations from simple Markdown text.
  2. PPTX Enhancer: Enhance existing PowerPoint presentations with AI-generated images.

Note: SlideUp uses AI technologies to generate both speaker notes and images, making it an intelligent presentation assistant.

Markdown to PPTX Web App

Getting Started

This tool allows you to create professional PowerPoint presentations from Markdown text. The conversion process is straightforward:

  1. Enter Markdown text in the text area or upload a Markdown file.
  2. Select options for adding speaker notes and/or images.
  3. Click "Generate PPTX" to start the conversion process.
  4. Once complete, download the generated PPTX file.

Markdown Format

The Markdown to PPTX tool expects a specific format to generate well-structured presentations:

# Main Presentation Title ### Slide 1: Title Slide - **title:** Your Presentation Title - **subtitle:** Optional Subtitle - **presenter:** Your Name - **date:** Presentation Date ### Slide 2: Content Slide - **text:** * First bullet point * Second bullet point * Third bullet point with **bold** text

Advanced Settings

The advanced settings section provides additional customization options:

  • API Keys: You can provide your own API keys for OpenAI, Stability AI, and DeepSeek.
  • Image Generation Prompt: Customize the prompt used for generating images.
  • DeepSeek Prompt Engineer Instructions: Fine-tune instructions for the DeepSeek AI.
  • Speaker Notes Generation Prompt: Customize how speaker notes are generated.

PPTX Enhancer Web App

Getting Started

The PPTX Enhancer allows you to upload existing PowerPoint presentations and enhance them with AI-generated images:

  1. Select and upload a PPTX file from your computer.
  2. Configure advanced settings if needed (API keys, image prompts).
  3. Click "Upload & Enhance" to start the process.
  4. Once complete, download the enhanced PPTX file.

How Enhancement Works

The enhancement process performs the following steps:

  1. Extracts text content from each slide.
  2. Generates relevant abstract images based on the text content.
  3. Adds images to slides in an alternating left-right layout.
  4. Adjusts text positioning and formatting to accommodate images.
  5. Creates a new enhanced PPTX file for download.

Note: The enhancement process works best with text-heavy slides and may take several minutes depending on the number of slides.

Under the Hood: How SlideUp Works

Architecture Overview

SlideUp is built with a modular architecture consisting of several Python components:

Component Description
markdown2pptx_webapp.py Flask web application that handles the Markdown to PPTX conversion and serves both web interfaces.
md2pptx.py Core module for processing Markdown and generating PPTX files.
pptx2enhancedpptx.py Handles enhancement of existing PPTX files by adding AI-generated images.
env_utils.py Utility functions for handling environment variables and API keys.

Technologies Used

SlideUp leverages several key technologies:

  • Python-PPTX: Library for creating and modifying PowerPoint files
  • Flask: Web framework for the user interface
  • OpenAI API: For generating speaker notes and enhancing content
  • Stability AI: For generating abstract images for slides
  • DeepSeek AI: For intelligent prompt engineering
  • Google Drive API: For optional file storage in Google Drive

Image Generation Process

The image generation process is particularly sophisticated:

  1. Text content from a slide is extracted and analyzed.
  2. DeepSeek AI creates an optimized prompt based on the slide content.
  3. The optimized prompt is sent to Stability AI for image generation.
  4. The generated image is processed and integrated into the slide.

Speaker Notes Generation

When speaker notes generation is enabled:

  1. The system extracts key points from each slide.
  2. These points are sent to OpenAI for expansion into comprehensive speaking notes.
  3. The AI generates natural-sounding, contextually relevant notes.
  4. These notes are added to the corresponding slides in the presentation.
